## 🤖 8 СПЕЦИАЛИЗИРОВАННЫХ АГЕНТОВ

### 1. 👔 Chief Career Coach (Главный Коуч)
- **Файл:** `agents_chief_coach.py`
- **Роль:** Координатор и стратег
- **Функции:**
  - Сбор информации о пользователе
  - Координация работы других агентов
  - Разработка персонального плана
  - Мониторинг прогресса

### 2. 📄 Resume Expert (Эксперт по Резюме)
- **Файл:** `agents_resume_expert.py`
- **Роль:** Специалист по оптимизации
- **Функции:**
  - Анализ текущего резюме
  - Оптимизация для ATS-систем
  - Адаптация под конкретные вакансии
  - Создание вариантов резюме

### 3. 🔍 Job Hunter (Охотник за Вакансиями)
- **Файл:** `agents_job_hunter.py`
- **Роль:** Специалист по поиску
- **Функции:**
  - Поиск подходящих вакансий
  - Анализ требований должности
  - Ранжирование по приоритету
  - Исследование компаний

### 4. ✅ Verification Specialist (Верификатор)
- **Файл:** `agents_verification_specialist.py`
- **Роль:** Контроль качества
- **Функции:**
  - Проверка грамматики и орфографии
  - Валидация согласованности информации
  - QA всех материалов
  - Верификация фактов

### 5. 🎯 Skills Coach (Коуч по Навыкам)
- **Файл:** `agents_skills_coach.py`
- **Роль:** Советник по развитию
- **Функции:**
  - Анализ пробелов в компетенциях
  - Рекомендации по обучению
  - Подбор сертификаций
  - Планирование развития

### 6. 🗣️ Interview Mentor (Интервьюер-Ментор)
- **Файл:** `agents_interview_mentor.py`
- **Роль:** Тренер по интервью
- **Функции:**
  - Подготовка STAR-структурированных ответов
  - Поведенческая подготовка
  - Практикование интервьюирования
  - Стратегия переговоров

### 7. 📊 Market Analyst (Аналитик Рынка)
- **Файл:** `agents_market_analyst.py`
- **Роль:** Исследователь рынка
- **Функции:**
  - Анализ тенденций рынка труда
  - Исследование уровней зарплат
  - Анализ требуемых навыков
  - Прогнозирование развития

### 8. 🌟 Branding Strategist (Стратег Персонального Бренда)
- **Файл:** `agents_branding_strategist.py`
- **Роль:** Специалист по личному бренду
- **Функции:**
  - Разработка стратегии персонального бренда
  - Оптимизация профиля LinkedIn
  - Создание привлекательного портфолио
  - Развитие онлайн-присутствия
